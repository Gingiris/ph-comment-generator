# 🚀 PH Comment Generator

Generate professional Product Hunt maker comments in seconds.

**[👉 Try it now](https://gingiris.github.io/ph-comment-generator/)**

## Why this tool?

Your maker comment is the first impression on Product Hunt. A well-crafted comment can:
- Set the tone for your launch
- Highlight key features clearly
- Encourage engagement and upvotes

This tool is based on **30x daily #1** Product Hunt launch experience from the [Gingiris Launch Playbook](https://github.com/Gingiris/gingiris-launch).

## Features

- ✅ **3 comment versions** - Story-driven, Feature-focused, Concise
- ✅ **4 tone options** - Friendly, Professional, Excited, Humble
- ✅ **Character count** - Optimized for 300-500 chars (best performing)
- ✅ **One-click copy** - Copy to clipboard instantly
- ✅ **Pro tips** - Best practices for launch day
- ✅ **Works offline** - No backend, no data collected

## Usage

1. Enter your product info
2. Click "Generate Comments"
3. Choose your favorite version
4. Copy and paste to Product Hunt

## Tech Stack

- Pure HTML/CSS/JS
- No dependencies
- No backend required
- GitHub Pages hosted
